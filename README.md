# 📊 Sales Analytics Dashboard

Interactive Streamlit dashboard for real-time sales analysis.

## Features
✅ Real-time filters (date, product, city)
✅ 5 KPI cards + interactive charts
✅ Data tables with CSV export
✅ Deploy to Streamlit Cloud

## Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Files
- app.py - Streamlit app
- sales_data.csv - Monthly sales data
- local_sales_data.csv - Local sales data
- requirements.txt - Dependencies

## Deploy
1. Push to GitHub
2. Connect to Streamlit Cloud
3. Get live URL
